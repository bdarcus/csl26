Reimagined for the Web

Significant evolution over CSL 1.0, targeting modern scholarly needs.

Built using a hybrid migration strategy: XML options extraction (87-100% citation accuracy) + output-driven template inference + hand-authored top styles. Learn more.

event

Advanced Dates

Handle complex historical ranges, uncertain dates, and non-Gregorian calendars with native precision.

translate

Multilingual Support

First-class support for multilingual citations, RTL languages, and locale-aware title casing.

title

Title Formatting

Distinct title/subtitle formatting and declarative rules for text transformations.

sync_alt

Type-Specific Overrides

Declare how components should render differently for each reference type (e.g., suppress publisher for articles but show for books) without procedural conditionals.

auto_awesome

Smart Presets

Top dependent styles decomposed into "presets" – an evolution of the 1.0 macro idea for better reusability.

memory

Rust-Native Styles

Core styles live as Rust code inside the processor for maximum performance, while keeping configuration accesssible.

link

Declarative Links

Configure links declaratively, such as automatically attaching DOI links to titles.

build

Integrated Repo

Code, schema, and documentation all in one maintained repository for easier iteration.

speed

Binary Performance

Native CBOR support for lightning-fast loading of massive bibliographies and styles.

CLI First Workflow.

Designed for developers and automation. Generate citations directly from your terminal or integrate into your build pipeline.

Process a bibliography

$ csln process inputs/paper.json styles/apa.yaml

Convert to binary (Performance Mode)

$ csln convert styles/apa.yaml --output styles/apa.cbor

Validate your style

$ csln validate styles/my-custom-style.yaml
